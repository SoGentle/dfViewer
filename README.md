# dfViewer
A cross-platform PyQt GUI for visualizing pandas dataframes in table format.  Implements some pandas operations,<br>
and includes a GUI for basic plotting with matplotlib.

#### Requirements:
Python 3.x<br>
pandas<br>
numpy<br>
matplotlib<br>
PyQt4<br>
Or the Anaconda distribution<br>
